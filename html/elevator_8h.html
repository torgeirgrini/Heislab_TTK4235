typedef struct Elevator Elevator

An Elevator struct which holds a queue system, memory variables and control variables.

Parameters
queue_matrixA matrix that holds a value for every type of order at every floor for the elevator. The values are non-zero for an active order and zero for a none-active order
current_floorHolds the value for the current or last visited floor
previous_floorHolds the value for the previously visited floor
current_movementHolds the value for the current direction of the elevator
previous_directionHolds the value for the previous direction of the elevator
order_directionHolds the value for the order type of the last handled order
current_stateHolds the current state for the elevator
stop_light_setA truthy value for if the stop light is set
timer_setA truthy value for if the timer has been set
